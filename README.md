# OtelSitesi 🌐

**OtelSitesi**, kullanıcıların oteller hakkında bilgi alabilecekleri, rezervasyon yapabilecekleri ve otel yönetimlerinin müşterilere yönelik hizmetlerini kolayca yönetebilecekleri modern bir web uygulamasıdır.  
Bu proje, kullanıcı dostu bir arayüz ve güçlü bir altyapıyla geliştirilmiştir. Hem otel müşterilerinin hem de otel yönetimlerinin ihtiyaçlarını karşılayacak şekilde tasarlanmıştır.

# Ana Sayfa

![Ana Sayfa Görüntüsü 1](./screenshouts/AnaSayfa1.png)  
![Ana Sayfa Görüntüsü 2](./screenshouts/AnaSayfa2.png)

### **Ana Sayfa Özellikleri**
- Kullanıcı dostu arayüz ile otel bilgilerine kolay erişim.
- Responsive tasarım: Tüm cihazlarla uyumlu.
- Tek tıkla rezervasyon sayfasına erişim.
- Harita entegrasyonu ile otel konumu ve iletişim bilgilerine hızlı erişim.
